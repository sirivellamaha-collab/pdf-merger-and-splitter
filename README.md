Project Name: PDF Merger & Splitter Pro

## Project Scope

PDF Merger & Splitter Pro is a desktop application developed using Python 3.10.11. This project helps users combine multiple PDF files into a single PDF and split PDF files into separate pages. The main goal of this project is to make basic PDF management easier through a simple graphical user interface.

I developed this project during my internship to improve my practical knowledge of Python programming and GUI development.
While working on this project, I learned how to work with PDF files, create a graphical user interface using CustomTkinter, organize code into different modules, and manage project files properly.
This application provides a simple way to perform PDF operations without using any online tools or additional software.

## Installation

1. Download or clone the repository.

2. Open Command Prompt in the project folder.

3. Install the required libraries using:

pip install -r requirements.txt

4. Run the application using:

python src/main.py

## Usage

1. Open the application.
2. Select the PDF files you want to work with.
3. Choose the required operation.
4. Save the generated output file.
5. Check the output folder to view the generated files.
